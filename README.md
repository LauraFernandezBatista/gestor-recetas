# 🥘 Gestor de Recetas

Aplicación web moderna desarrollada con **React**, **Ionic** y **Vite**,
que permite gestionar un recetario personal de forma sencilla: añadir,
editar, buscar y eliminar recetas.\
Un proyecto pensado para demostrar buenas prácticas de desarrollo
frontend y organización de código.

------------------------------------------------------------------------

## 🚀 Características

-   📋 Listado de recetas con búsqueda por título, ingredientes o
    etiquetas.\
-   ➕ Añadir nuevas recetas mediante un formulario intuitivo.\
-   ✏️ Editar recetas existentes.\
-   ❌ Eliminar recetas con confirmación de seguridad.\
-   📱 Interfaz adaptable (responsive) gracias a **Ionic**.\
-   💾 Persistencia en **LocalStorage**, sin necesidad de backend.

------------------------------------------------------------------------

## 🛠️ Tecnologías utilizadas

-   [React](https://react.dev/)\
-   [Ionic React](https://ionicframework.com/docs/react)\
-   [React Router DOM](https://reactrouter.com/)\
-   [Vite](https://vitejs.dev/)
